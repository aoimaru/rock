# Rock 簡易的なテスト環境の雛形管理リポジトリ EC2インスタンス関連(お金のかかるサービス)
## 必要な準備
1. orbitを利用したグローバルリソースの作成
2. forgeを利用した準グローバルリソース(ネットワーク関連)の作成
    * ただし、NATゲートウェイ, SSMエンドポイントはお金がかかるサービスなので, `rock`で作成

## 引数
* プロジェクト名
* 環境 (dev/stg/prod) など
* バージョン v1.0.0 など

## 作成するリソース
* アプリケーション配置用のEC2インスタンス
* 監視(攻撃)サービス配置用のEC2インスタンス
* SSM接続用エンドポイント
    * 監視(攻撃)サービス配置用のEC2インスタンスにはSSM経由で通信を行うため
* NATゲートウェイ
    * アプリケーション配置用のEC2インスタンスから, 外部へリクエストを投げるため
